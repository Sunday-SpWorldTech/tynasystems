Tools We Use

Notion, ClickUp, CRM Tools and Web Development Tools power the Tyna Systems stack.

This page shows the high-value tools used to document work, execute projects, manage leads, build websites and keep users organized inside their dashboards.

Why these tools?

They are simple enough for small teams and strong enough to build a real operating system for sales, finance, delivery, SOPs, reporting and client follow-up.

  • Notion: documentation, SOPs, internal knowledge, client hubs and command centers.
  • ClickUp: task execution, ownership, deadlines, project status and team accountability.
  • CRM Tools: lead capture, client records, follow-up reminders, sales pipeline visibility and relationship tracking.
  • Web Development Tools: VS Code, React, Node, Express, MongoDB, UI/UX, GitHub and Render deployment for professional websites and backend systems.
